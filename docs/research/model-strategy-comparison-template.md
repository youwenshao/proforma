# Model Strategy Comparison Template

## Strategies

- Firm-specific simulated models by firm tier.
- Matter-type-specific simulated models.
- Pooled global model.
- Leave-one-tier-out evaluation.

## Required Outputs

- Accuracy by strategy.
- Calibration by strategy.
- Sample size per segment.
- Minimum estimated records per firm for useful models.
- Segments where pooled data improves accuracy.
- Privacy and solicitor confidentiality caveats.

## Governance Language

Pooled anonymized training remains legally gated until data sharing,
confidentiality, anonymization, PDPO, and data residency approvals are complete.
